HTML/CSS Cheatsheet

Tables

Table Tags

Tag Name Description 
<table>
 Table Wrapper element for HTML tables 
<thead>
 Table Head Defines the row(s) containing column headers 
<tbody>
 Table Body Defines the row(s) containing actual table data 
<tr>
 Table Row Container for an individual row 
<td>
 Table Data Container for an individual cell 
<tfoot>
 Table Foot Defines the row(s) containing the footer

Table Attributes

Attribute Name Description 
colspan
 Column Span Defines how many columns a td element should span 
rowspan
 Row Span Defines how many rows a td element should span

Common CSS Properties

Property Description 
font-family
 Changes the font/typeface 
font-size
 Changes the size of the font/typeface. Set in pixels, em, rem, percentages, or relative/absolute keywords (ie medium/smaller) 
font-weight
 Controls how bold or thin text appears. Set in relative keywords (bolder, normal) or a number between 1 and 1000 
text-align
 Controls where text element appears on the page. Default is left 
color
 Changes the font color. Can be set in hex, rbg, hsl, or named colors. 
background-color
 Changes the background color of an element 
background-img
 Changes the background to an image provided with 
url("example")
. Can be a relative file path or a link to an external file		 
opacity
 Changes the opacity of an element. Set to a number between 0.1 and 0.99 or a percentage 
list-style
 Can change the style of a bulleted/unordered list (ie from round to square bullets with the value 
square
)
